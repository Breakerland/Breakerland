## Liste des codes format
| Format name       | Chat format code |
| :----------------:|:----------------:|
| Obfuscated        | &k               |
| **Bold**          | &l               |
| ~~Strikethrough~~ | &m               |
| Underline         | &n               |
| *Italic*          | &o               |
| Reset             | &r               |

Pour les quêtes :
Mise en forme nom &0[&2Nom NPC&r] : Texte
Message alerte pré-requis non remplis : &[&cAttention&r] : &cPrè-Requis non remplis, il te faut &rX&c point de quêtes
